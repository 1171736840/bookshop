# bookshop
这个网上书城小项目是我在课余时间写出来锻炼技术的，前端使用了jQuery框架，后端使用了Java语言，Spring、SpringMVC和Hibernate框架，MSSQLServer数据库。 目前在Chrome、Edge、IE9+等浏览器上经过测试正常运行。
